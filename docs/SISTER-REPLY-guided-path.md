**Subject:** Re: AI assessment & therapy recommendation — how it maps to SoulCap

Thank you — this is exactly the clinical shape we want users to feel: notice what’s going on → match something useful → try it → learn over time. You’re describing a good stepped-care self-help flow.

## What SoulCap already ships (offline, private — nothing leaves the phone)

| Your idea | Already in SoulCap |
|---|---|
| Select how they’re feeling | **Now** check-in (Steady / Wired / Flat / Heavy) |
| Brief interactive assessment | Calm “what do you need?” + **What’s happening?** experiences |
| Anxiety / body / worry symptoms | 24 experiences (racing heart, catastrophising, rumination, etc.) |
| Cognitive distortions | Thought record, defusion, worry-vs-problem exercises + cognitive experiences |
| CBT / DBT / ACT / BA techniques | 37 exercises with psychoeducation sources — framed as **skills**, not “you are in therapy” |
| Practical exercises | Breathing, 5-4-3-2-1, PMR, BA, opposite action, values check, journaling |
| Progress over time | Journey / seven-day / patterns / timeline — **facts only**, no severity score |
| Crisis / severe distress | Hard-coded **Help** screen + PHQ-9 item-9 routing (no crisis phone numbers by product decision — we can’t promise any line is reachable) |
| Privacy | `localStorage` only; zero network after load |

## What we will not ship as written

- Cloud / LLM “AI therapist” chat  
- Labelling people with disorders or saying “you need CBT / DBT / ACT”  
- Symptom **severity scores** / streak-style dashboards  
- Country-specific crisis hotline lists  

## What we’re building next (your review welcome)

1. A short **Guided Path**: arrival feeling → a few symptom chips → plain-language “why this family of exercises may help” → one tap to begin.  
2. Educational footnotes only: “Some people learn related ideas in CBT / DBT / ACT / behavioural activation. SoulCap is not those therapies.”  
3. You review the chip list + “why” copy; clinical banner stays until licensed sign-off.
