# SoulCap — AI Emotional Wellness Platform

SoulCap is a clinical-grade AI emotional wellness platform powered by the **Living Mind Model (LMM)** — a dynamic psychological profile that evolves with every interaction to deliver deeply personalized, emotionally intelligent support.

---

## What It Does

- **Living Mind Model** — Tracks 30+ psychological traits (resilience, self-compassion, emotional regulation, attachment style, and more) using Exponential Moving Average updates. Decays and consolidates over time like a real mind.
- **10-Step AI Orchestration** — Every message goes through: quick scan → emotion analysis → safety assessment → LMM retrieval → memory retrieval → strategy selection → prompt assembly → model routing → response generation → hard rail check.
- **4-Tier Safety System** — NONE / DISTRESS / ELEVATED / ACUTE. Tier 3 (ACUTE) returns a hardcoded crisis response, bypassing the LLM entirely and alerting your team.
- **Semantic Memory Engine** — pgvector-powered episodic and semantic memory. The AI remembers what matters across sessions.
- **Panic Mode** — Grounding exercises (5-4-3-2-1, box breathing, safe place visualization, cold water reset), crisis resources, and safety escalation.
- **Voice Notes** — OpenAI Whisper transcription with S3 storage.
- **Pattern Detection** — Automatic psychological pattern recognition via Claude Haiku tool_use.
- **Insight Generation** — Weekly insight synthesis with deduplication.
- **Growth Reports** — Monthly narrative growth reports generated by Claude Sonnet.
- **Check-Ins** — Daily emotional check-ins (valence, arousal, groundedness, energy) with LMM updates.
- **Goals & Habits** — Goal tracking with milestone detection; habit streaks with LMM reinforcement.
- **Journaling** — AI-companioned journal entries with emotional state extraction.
- **Multi-Tenant** — Organization-scoped with Clerk authentication.
- **Push Notifications** — APNs direct integration.
- **Billing** — Stripe subscriptions.

## Getting Started

### Prerequisites

- Node.js 20+
- PostgreSQL 15+ with pgvector extension
- Redis 7+
- Clerk account
- Anthropic API key
- OpenAI API key (for Whisper)
- AWS S3 bucket

### Backend

```bash
cd backend
cp .env.example .env
# Fill in your environment variables
npm install
npx prisma migrate dev
npm run start:dev
```

### Mobile

```bash
cd mobile
cp .env.example .env
npm install
npx expo start
```

### Environment Variables

```env
# Database
DATABASE_URL=postgresql://...

# Auth
CLERK_SECRET_KEY=sk_...
```

## Safety & Ethics

SoulCap is **not a replacement for professional mental health care**.

- The AI never claims to be a therapist
- The AI never provides medical diagnosis
- Crisis signals trigger immediate escalation to hardcoded safety resources
- All message content is encrypted at rest
- Safety flags are logged without storing raw message content
- Users are always encouraged to seek professional help when appropriate

**Crisis Resources always available:**
- 988 Suicide & Crisis Lifeline (call or text 988)
- Crisis Text Line (text HOME to 741741)
